for creating an env (first time) :

1. pip install virtualenv
2. virtualenv <name of your env>
3. cd env
4. cd .\Scripts\
5. activate
6. come to root folder
7. python app.py

HOW TO RUN 

1. cd env
2. cd .\Scripts\
3. activate
4. come to root folder
5. python app.py
